## ABOUT
The first project at 42 school, libft, involves learning how the standard functions of C programming work by writing them from scratch and creating a personal library.

<br>

## HOW TO USE
1º - To compile libft you should run `a.out` in the terminal.


2º - Run the program using `a.out`
```sh
$ ./a.out
```

<br>

## DISCLAIMER
At 42 School, it is expected that almost every project is written in accordance with the Norm, which is the coding standard of the school:

	- No for, do while, switch, case or goto are allowed
	- No more than 25 lines per function and 5 functions per file
	- No assigns and declarations in the same line (unless static)
	- No more than 5 variables in 1 function
	...
